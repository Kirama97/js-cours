# 📚 Cours JavaScript — Série d'exercices

Bienvenue dans ce cours complet de JavaScript conçu pour les étudiants débutants à intermédiaires.

## 🗂️ Structure du projet

```
js-cours/
├── index.html                    ← Interface visuelle du cours (ouvrir dans un navigateur)
├── README.md                     ← Ce fichier
└── modules/
    ├── 01-variables-types.js     ← Variables, types, opérateurs
    ├── 02-conditions.js          ← if/else, switch, ternaire
    ├── 03-boucles.js             ← for, while, for…of, for…in
    ├── 04-tableaux.js            ← Arrays, map, filter, reduce
    ├── 05-fonctions.js           ← Fonctions, closures, récursivité
    ├── 06-objets.js              ← Objets, classes, héritage
    ├── 07-dom-evenements.js      ← DOM, événements, localStorage
    └── 08-asynchrone.js          ← Callbacks, Promises, async/await
```

## 🚀 Comment utiliser ce cours

### Option 1 — Interface web (recommandé)
1. Ouvrez `index.html` dans votre navigateur
2. Explorez les 8 modules depuis la page d'accueil
3. Utilisez le **Playground** intégré pour tester du code directement

### Option 2 — Fichiers JS directement
1. Ouvrez un fichier `modules/0X-nom.js` dans VS Code
2. Lisez les **leçons commentées**
3. Complétez les exercices marqués `// Votre code ici ↓`
4. Exécutez avec Node.js : `node modules/03-boucles.js`

## 🎯 Types d'exercices

- **EX (numérotés)** — Exercices guidés avec consignes précises
- **🔥 DÉFI** — Exercices ouverts plus complexes, pour aller plus loin

## 🛠️ Outils recommandés

- **Éditeur** : [Visual Studio Code](https://code.visualstudio.com/)
- **Extension** : Quokka.js (exécution inline dans VS Code)
- **Runtime** : [Node.js](https://nodejs.org/) pour exécuter en terminal
- **Navigateur** : Chrome ou Firefox (DevTools > Console)

## 💡 Conseils pour les étudiants

1. **Ne pas copier-coller** — Tapez le code vous-même pour muscler la mémoire
2. **Tester les erreurs** — Provoquez des erreurs exprès pour comprendre les messages
3. **Modifier les exemples** — Changez les valeurs, observez les effets
4. **Chercher avant de regarder** — Essayez 30 min avant de chercher la solution
